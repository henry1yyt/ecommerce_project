# 数据字典

## 原始与扩展字段

| 字段 | 含义 | 类型/示例 |
|---|---|---|
| order_id | 订单编号 | AI00000001 |
| user_id | 用户编号 | U01163 |
| user_gender | 用户性别 | 男/女 |
| user_age | 用户年龄 | 18～65 |
| age_group | 年龄段 | 23-30 |
| user_level | 会员等级 | 新客、普通、学生、银卡、金卡 |
| city / province / region | 城市、省份、大区 | 沈阳、辽宁、东北 |
| product_id / product_name | 商品编号和名称 | P007、办公椅 |
| category | 商品品类 | 电脑办公、服饰运动等 |
| brand / supplier | 品牌和供应商 | 文本字段 |
| price | 商品单价 | 数值 |
| cost | 商品单位成本 | 数值 |
| quantity | 购买数量 | 1～8 |
| discount | 折扣系数 | 0.65～1.00 |
| coupon_amount | 优惠券金额 | 数值 |
| payment_method | 支付方式 | 支付宝、微信支付等 |
| sales_channel | 销售渠道 | 网页商城、手机APP、直播间等 |
| traffic_source | 流量来源 | 自然搜索、直播推荐等 |
| campaign_name | 活动名称 | 日常、618大促、双十一 |
| device_type | 设备类型 | iOS、Android、PC等 |
| order_date / order_hour | 下单日期和小时 | 日期、8～23 |
| is_returned | 是否退货 | 0/1 |
| return_reason | 退货原因 | 未退货、尺码不合适等 |
| delivery_days | 实际配送天数 | 0～8 |
| customer_rating | 客户评分 | 1～5 |
| ab_group | A/B 分组 | A/B |
| shipping_fee | 物流费用 | 数值 |
| marketing_cost | 营销成本 | 数值 |

## 清洗后派生字段

| 字段 | 计算口径 |
|---|---|
| sales_amount | 单价 × 数量 × 折扣 − 优惠券 |
| refund_amount | 销售额 × 是否退货 |
| net_sales_amount | 销售额 − 退款金额 |
| product_cost | 单位成本 × 数量 |
| gross_profit | 净销售额 − 商品成本 |
| reverse_logistics_cost | 退货订单固定计入 8 元逆向物流成本 |
| net_profit | 毛利润 − 物流费 − 营销成本 − 逆向物流成本 |
| gross_margin | 毛利润 ÷ 净销售额 |
| net_margin | 净利润 ÷ 净销售额 |
| is_weekend | 是否为周末，0/1 |
| month | 订单月份，YYYY-MM |

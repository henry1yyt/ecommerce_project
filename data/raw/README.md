# 原始数据

唯一数据源：和鲸社区 `online-retail-ii（英国礼品零售商数据）`

https://www.heywhale.com/mw/dataset/6a124df17e367d3a68e4c96b

下载后将文件命名为 `online_retail_II.xlsx` 并放在本目录。原始文件约 43.5 MB，
含 2009—2010 与 2010—2011 两张年度工作表。分析脚本读取全部工作表，不抽样、
不生成数据、不进行 AI 扩展。

上游来源：Online Retail II / UCI Machine Learning Repository，CC BY 4.0。
